Kyo
---
>Kyo feature tests using Scala 3.

Notes
-----
* **Tries** were removed in version *0.8.7*.
* **Seqs** were removed in version *0.9.2*.

Build
-----
1. sbt clean compile

Test
----
1. sbt clean test

Run
---
1. sbt clean run
